# Trees

Generating trees in terminal

## Examples

`node tree 1`
```
    *
   ***
  *****
 *******
    |
```

`node tree 2`
```
         *
        ***
       *****
      *******
       *****
      *******
     *********
    ***********
   *************
         |
         |
```

`node tree 3`
```
              *
             ***

           *******
          *********
         ***********
        *************
         ***********
        *************
       ***************

     *******************

             |||
```
