# Trees

Generating christmas trees in terminal

## Examples

`node tree` generates a random tree

Or you can choose the size of the tree with :

`node tree 1`
```
    *
   ***
  *****
 *******
    |
```

`node tree 2`
```
         *
        ***
       *****
      *******
       *****
      *******
     *********
    ***********
   *************
         |
         |
```
